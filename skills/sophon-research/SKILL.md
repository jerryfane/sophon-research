# Sophon Research

Use this skill when the user asks to research AI evaluations, models, tools,
leaderboards, capabilities, organizations, people, or papers through Sophon's
public catalog.

Prefer the local `sophon-research` CLI when available. Sophon is read-only and
does not require an API key.

Do not use Sophon during benchmark-solving contexts where internet access is
disallowed. Treat Sophon as a discovery source; verify important technical
claims against primary papers or official documentation when accuracy matters.
